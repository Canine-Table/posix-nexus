\section{Object}
\label{Object}
\begin{NexMainBox}[light, hdrA, sdwA, crnA, grwB, secA]
	\begin{NexMainBox}[dark, crnA]]
		The following functions provide powerful object manipulation utilities, enabling reflection, property checking, and dynamic assignment operations for JavaScript objects.
	\end{NexMainBox}
	\begin{NexMainBox}[dark, crnA]
		\begin{NexListDark}
			\NexItemDark{\NexFunction{Obj.reflect(val)}: Inspects the given value \NexFunction{val} and retrieves the keys of its prototype if it is an object. If the prototype contains only one property, retrieves the object's own keys instead.}
			\NexItemDark{\NexFunction{Obj.methods(val)}: Logs all methods of the given object \NexFunction{val}, excluding common non-functional properties such as \texttt{length}, \texttt{name}, \texttt{prototype}, and others.}
			\NexItemDark{\NexFunction{Obj.isProp(val, prop)}: Checks if the given property \NexFunction{prop} exists within the reflected properties of the value \NexFunction{val}. Returns \texttt{true} if found, otherwise \texttt{false}.}
			\NexItemDark{\NexFunction{Obj.assign(val, prop, obj)}: Dynamically assigns properties from \NexFunction{obj} to \NexFunction{val}. If \NexFunction{prop} is defined, assigns properties to the specific sub-object at \NexFunction{val[prop]}; otherwise, assigns directly to \NexFunction{val}.}
		\end{NexListDark}
	\end{NexMainBox}
\end{NexMainBox}
