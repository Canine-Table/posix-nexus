\section{Network Protocols}
\label{Network Protocols}
\begin{NexMainBox}[light, hdrA, sdwA, crnA, grwB, secA]
	\begin{NexMainBox}[dark, crnA]
	Network protocols define the rules and conventions for communication between devices in a network. They ensure data is transmitted accurately, securely, and efficiently. 
	\end{NexMainBox}
	\begin{NexMainBox}[dark, crnA]
		\begin{NexListDark}
			\NexItemDark{\NexLink{Application Layer Protocols}{Application Layer Protocols}: Protocols that support end-user services, such as HTTP, SMTP, FTP, and DNS.}
			\NexItemDark{\NexLink{Transport Layer Protocols}{Transport Layer Protocols}: Protocols like TCP and UDP that handle data transfer between devices.}
			\NexItemDark{\NexLink{Network Layer Protocols}{Network Layer Protocols}: Protocols like IP and ICMP that define how data is routed across networks.}
			\NexItemDark{\NexLink{Data Link Layer Protocols}{Data Link Layer Protocols}: Protocols that manage data exchange over a physical link, such as Ethernet and PPP.}
		\end{NexListDark}
	\end{NexMainBox}
\end{NexMainBox}
