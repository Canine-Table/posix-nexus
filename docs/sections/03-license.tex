\label{sec:license:sub:compatable}
The following licenses are compatible with GPLv3, meaning that you can combine or link code under these licenses with GPLv3 code:
\bigskip
\begin{baseBoxOne}{}{dark}
    \begin{posnexItemize}
        \item[\sA] \sE{Apache License 2.0}: As long as the combined work adheres to the terms of the GPLv3.
        \item[\sA] \sE{MIT License}: Allows for great flexibility in how the software is used and distributed.
        \item[\sA] \sE{BSD License}: Both the 2-Clause and 3-Clause BSD Licenses are compatible with GPLv3.
        \item[\sA] \sE{Mozilla Public License 2.0}: Compatible with GPLv3, though care must be taken to comply with both licenses' requirements.
        \item[\sA] \sE{Creative Commons Zero (CC0)}: A public domain dedication that is compatible with GPLv3.
