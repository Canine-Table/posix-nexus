Activity 4 Box Model & Pseudo Selectors

Overview

In this pen practice using the box model to grant elements the space they need and select items without creating new classes.

Guidelines

The guidelines are simple:

  • Edit the HTML where indicated. (comments will indicate where to add code). Edit the CSS where indicated.
  • You may (probably will) have to visit W3Schools to do some research!

Task 1

Instructions: Create a short navigation with 3 items. Style the middle item using a pseudo class.

Task 2

Instructions: Create a list of 4 colours. Style each one with their corresponding colour using pseudo classes.

Task 3

Instructions: Create two sections with paragraphs inside. Make them both sit next to eachother on the same line. Add a different background colour for each section. Give the paragraphs inside some space by adding some padding.

Task 4

Instructions: Create an ordered list of your favourite hobbies. Style every even item in that list using one pseudo class.

Task 5

Instructions: Create a heading. Use pseudo elements to add an emoji before and after the heading.

Task 6

Instructions: Play the css diner game and write in a paragraph what level you were able to get to. Click here

Challenge Task 7

Instructions: Create a navbar with 5 links (can be placeholders). Create a hover style and an focus style for these links.

Challenge Task 8

Instructions: Create a paragraph. Using pseudo elements, change the style of the first letter, the first line and when text in highlighted by the user.

Activity 4 Responsiveness and Media Queries

Overview

In this pen practice making elements responsive.

Guidelines

The guidelines are simple:

  • Edit the HTML where indicated. (comments will indicate where to add code). Edit the CSS where indicated.
  • You may (probably will) have to visit W3Schools to do some research!

Task 1

Instructions: Create 3 paragraphs. Make them sit into 3 equal columns using % units.

Task 2

Instructions: Add an image. Make it span the full width of its container without squishing or stretching (aka aspect ratio needs to stay the same.)

Task 3

Instructions: Create a description list (new tags, look them up!) and describe 4 different animals. Create a media query for 600px. At a screen size of 600px and larger have the list stack horizontally.

Task 4

Instructions: Create an h1. Then create 3 media queries for this h1 and have it change colour at each breakpoint.

Task 5

Instructions: Create a flexbox with 6 elements inside of it. Make one breakpoint (media query) for this flexbox and change at least two flex properties at that breakpoint.

Task 6

Instructions: Create a grid with at least 3 elements to fill it. Change the layout of this grid at a breakpoint of 40em.

Challenge Task 7

Instructions: Create a navbar with 5 links (can be placeholders). This navbar will need to behave like a real one. At mobile view the menu should be activatable using a hamburger icon and be stacked vertically. At desktop view the icon should disappear and the menu should stack itself horizontally. Hint: JS is necessary for this.

Challenge Task 8

Instructions: Make this whole activity page disappear at a size of 300px or smaller.
